================================================================================
                         TROUBLESHOOTING
================================================================================

Issue: Port already in use
Solution: Change port in docker-compose.yml to 8001:8000

Issue: Database connection failed
Solution: Wait a few seconds, entrypoint.sh handles this automatically

Issue: Container won't start
Solution: Check logs with: docker-compose logs web

Issue: Need fresh start
Solution: docker-compose down -v && docker-compose up --build
